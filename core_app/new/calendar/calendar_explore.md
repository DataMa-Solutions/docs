# 1. Calendar in Explore

In Datama Explore, calendars are used to enrich **data exploration and contextual analysis**.

They help users understand how events relate to KPIs during exploration.

<center><img src="{{site.url}}/{{site.baseurl}}/core_app/new/calendar/images/calendar_in_helper.png"/></center>

<br>

# 2. WebApp usage (Helpers)

When a calendar is attached to a workbook, its events become accessible in Datama Helpers.

At this stage:
  * Events are displayed only when a date dimension is used as one of the breakdown (drill-down) dimensions.
  * Only events that fall within the selected analysis period are shown.

This brings important operational context directly into the user assistance interface.


<br>

# 3. Light extension (Explore mode)

In the Light extension Explore mode:
- Calendars can be activated per analysis
- Events appear as contextual overlays in charts
- Users can visually correlate trends with events
