# 1. Calendar in Detect

In Datama Detect, calendars are used to **contextualize anomaly detection results**.

<br>

When a calendar is enabled:
- Events are overlaid on detected anomalies
- Users can see if anomalies align with known events
- Helps interpret possible root causes

<center><img src="{{site.url}}/{{site.baseurl}}/core_app/new/calendar/images/calendar_in_detect.png"/></center>

<br>

# 2. Scope usage

In Detect, the calendar scope is used only for filtering relevant events if you apply filter on the dimension map in the scope field of your calendar
Unlike Compare, it does not impact scoring.
