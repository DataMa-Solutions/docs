Une avancée majeure pour **toutes les personnes qui travaillent dans Excel** 🤓

Vous pouvez utiliser **Datama Compare directement dans votre classeur** 🎉

* Installez depuis la [fiche Microsoft AppSource](https://marketplace.microsoft.com/en-hk/product/WA200010671?tab=Overview), ou dans Excel via **Accueil** > **Compléments** et la recherche **Datama Compare** (si votre Microsoft 365 est géré par votre organisation, un administrateur peut devoir autoriser le complément).
* Préparez un tableau agrégé (KPI, dimension de comparaison, étapes, dimensions d’explication)—la même logique que dans nos autres extensions Compare.
* Ouvrez le volet du complément, sélectionnez votre plage de données, ajustez dimensions et métriques si besoin, puis lancez l’analyse.
* Option : démarrez un **essai payant** depuis le pied de page pour retirer le branding et enregistrer les réglages ; collez votre clé de licence dans **Paramètres de l’éditeur** comme décrit dans la doc.

➡️ Pourquoi les équipes l’adoptent :

* **Restez dans Excel :** expliquer les variations là où se trouvent les données—sans copier-coller vers un autre outil.
* **Visualisez directement vos scénarios :** modifiez des chiffres ou des hypothèses dans la feuille et **relisez l’impact** sur la waterfall et le KPI tree
* **Profitez de toutes les fonctionnalités Compare Light :** waterfall et KPI tree, scoring intelligent, mix effect, commentaires intelligents et exploration interactive.

🖱️ [En savoir plus dans la documentation Excel Compare](https://datama-solutions.github.io//docs/extensions/how-to-use/excel_compare.html)
