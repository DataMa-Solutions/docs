* **Datama Spreadsheet** embeds an **Excel-style grid** (formatting, formulas, pivot-style **Rows / Columns / Values**) inside supported BI dashboards, as a light extension for **Tableau**, **Power BI**, **Qlik**, and **Looker Studio**. Documentation: [Datama Spreadsheet]({{site.url}}/{{site.baseurl}}/extensions/datama-spreadsheet/introduction.html).
