* **Create Calendar event in Compare Light (extensions)**
  * You can now create an event from Datama Compare light extension by right clicking on a bar and then "Add event"; This was already possible in Datama Detect and Explore, and is now possible in Datama Compare, since Compare now takes into account events in Comments text, through Event interest (see previous release). 
  * Event will be logged in a calendar called "From [Viz tool]" (e.g. "From Tableau") in the web app and will be activated by default in the visual. 
  * You can then include that calendar as any other calendar in Datama Web app and light extensions 

* Multiple bugs and improvements have been shipped including
  * Translation of event interest comments in Compare in French
  * Default time zone management for day events in Calendar uploads
  * Multiple improvements of Axis display in Light extensions
  * Secondary comparison filtering in Compare Light extension
