* **Compare** — hovering a Mix segment on the [waterfall]({{site.url}}/{{site.baseurl}}/core_app/new/compare/interface/waterfall.html#31-mix-effect) now shows the segment's **average performance** to clarify the reason for mix effect.
* **Power BI** — extension now handles up to **200k rows** (vs. 30k) and **auto‑saves** configuration and licence: no more JSON copy‑paste in the "Config JSON" field. See [Power BI Compare]({{site.url}}/{{site.baseurl}}/extensions/how-to-use/power-bi_compare.html#5-getting-started).
* **Tableau** — Viz extension is now compatible with Tableau **2026.X**.
* **Bug fixes** — "Impact in points (pts)" display in [Compare]({{site.url}}/{{site.baseurl}}/extensions/datama-compare/settings.html#31-impact-in-points-pts), and reload of day window settings in [Detect]({{site.url}}/{{site.baseurl}}/extensions/datama-detect/settings.html#11-modeling).
