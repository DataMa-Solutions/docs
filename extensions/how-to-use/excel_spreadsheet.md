> Using Datama Spreadsheet as a Excel extension lets you run spreadsheet analysis and pivot workflows in your Excel sheets.

<br/>

# 1. Overview

This document explains how to use Datama Spreadsheet in Excel.

The extension is driven by pivot mapping:

- Rows
- Columns
- Values

<br/>

# 2. Main features

- Full spreadsheet interactions
- Formatting and conditional formatting
- Formulas and calculations
- Pivot-table output with filters and sorting
- Result mode for presentation-ready output

<br/>

# 3. Add the extension into your app

To use Datama Spreadsheet in Excel:

1. Add the Datama Spreadsheet extension package in Excel Management Console.
2. Open your app and switch to edit mode.
3. Enable advanced options if needed.
4. Drag and drop the Datama Spreadsheet extension from custom objects.
5. Map fields and configure Rows, Columns, and Values.

<br/>

# 4. Getting started

Recommended setup:

1. Add grouping dimensions in **Rows**.
2. Add split dimensions in **Columns**.
3. Add measures in **Values**.
4. Apply formulas, filters, sort, and formatting.
5. Save settings: Once you are happy with your settings, in the Excel visualization properties panel for the extension, paste the JSON automatically added to your clipboard in the "Config JSON" field. If you are unsure whether it is the latest version, click the Save button in the top-right corner to generate the updated JSON, then paste it again into the Config JSON field. Please note that no customized settings will be saved unless you complete this step.
6. Exit edit mode for result-only display (for example via the extension properties where **Edit mode** is toggled).

Result mode hides toolbar and grid lines.
