> Load a Datama Assess export HTML into the [HTML viewer by Datama]({{site.url}}/{{site.baseurl}}/extensions/how-to-use/powerpoint.html) add-in to display statistical significance analysis inside PowerPoint.

<br/>

# 1. Overview

The PowerPoint HTML viewer routes to Assess when the export configuration contains distribution module hints (`main.distribution` or `"distribution"` in the serialized configuration), or when the `DATAMA_SOLUTION_*` marker is set to `assess`.

<br/>

# 2. Export from Datama Assess

1. Build your Assess analysis in Datama (or a connected host).
2. Export as HTML with Datama marker blocks.
3. In the PowerPoint add-in, click **Load HTML file** and select the export.

<br/>

# 3. Interact with the visualization

- Toggle **Edit mode** in editable PowerPoint views to adjust settings.
- Reload another file with **Load HTML file** at any time.
