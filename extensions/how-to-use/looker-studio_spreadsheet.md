> Using Datama Spreadsheet in Looker Studio brings pivot-table and spreadsheet analysis to your report without leaving the dashboard.

<br/>

# 1. Overview

This document explains how to configure Datama Spreadsheet in Looker Studio.

The extension uses a pivot-table setup:

- Rows
- Columns
- Values

<br/>

# 2. Main features

- Spreadsheet cell operations and formatting
- Formulas and functions
- Sorting and filtering
- Pivot-table analysis in extension context
- Edit mode and result mode display behavior

<br/>

# 3. Add the extension into your report

To use Datama Spreadsheet in Looker Studio:

1. Open your Looker Studio report in edit mode.
2. Add a community visualization.
3. Select Datama Spreadsheet from your available Datama extensions.
4. Map the required dimensions and measures.
5. Configure Rows, Columns, and Values.

<br/>

# 4. Getting started

Suggested setup:

1. Choose dimensions for **Rows**.
2. Choose dimensions for **Columns**.
3. Choose measures for **Values**.
4. Configure formatting and calculations.
5. **[Paid user only] Save settings**: Once you are happy with your settings, in the Looker Studio side panel, in the "Style" tab, paste the JSON automatically added to your clipboard in the "Config JSON" field. If you are unsure whether it is the latest version, click the Save button in the top-right corner to generate the updated JSON, then paste it again into the Config JSON field. Please note that no customized settings will be saved unless you complete this step.

<center><img style="width:80%;" src="{{site.url}}/{{site.baseurl}}/extensions/how-to-use/assets/JSONsave.gif" /></center>

6. **[Paid user only] Leave edit mode**: In the Looker Studio side panel, in the "Style" tab, untick the "Edit mode" toggle when you want a clean spreadsheet output.

<center><img style="width:80%;" src="{{site.url}}/{{site.baseurl}}/extensions/how-to-use/assets/edit_mode.gif" /></center>

Result mode displays the spreadsheet result without toolbar and without grid lines.
