# 1. Overview

This document explains how to use the Datama Spreadsheet light extension in Tableau Viz.

Datama Spreadsheet is based on a pivot workflow where you assign fields to:

- Rows
- Columns
- Values

<br/>

# 2. Main features

- Spreadsheet interactions and formatting
- Formula and function support
- Filtering and sorting
- Pivot-table analysis directly in the extension
- Result mode to display only the spreadsheet output

<br/>

# 3. Add the extension into your report

To use Datama Spreadsheet in Tableau:

1. Open Tableau Desktop and create a worksheet.
2. In the Marks area, choose **Add extension**.
3. Use the `.trex` file provided by Datama Spreadsheet.
4. Drag and drop your dimensions and measures in the extension field mapping.
5. Build your pivot setup with Rows, Columns, and Values.

<br/>

# 4. Getting started

Recommended setup sequence:

1. Pick dimensions for **Rows**.
2. Pick dimensions for **Columns**.
3. Pick numeric fields for **Values**.
4. Apply formatting, formulas, and filters if needed.
5. Save settings: Click the **Save** button in the top-right corner of the Datama extension to persist your configuration in the workbook. 
6. Exit edit mode to show only the spreadsheet result in the dashboard (for example via **Format extension** in the Marks panel and the **Edit mode** toggle).

When edit mode is disabled, toolbar and grid lines are hidden.
