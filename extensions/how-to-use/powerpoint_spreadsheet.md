> Load a Datama Spreadsheet export HTML into the [HTML viewer by Datama]({{site.url}}/{{site.baseurl}}/extensions/how-to-use/powerpoint.html) add-in to display an Excel-like editable pivot table inside PowerPoint.

<br/>

# 1. Overview

The PowerPoint HTML viewer routes to Spreadsheet when the export configuration contains `main.sheet` or `luckysheet`, or when the `DATAMA_SOLUTION_*` marker is set to `spreadsheet`.

<br/>

# 2. Export from Datama Spreadsheet

1. Build your Spreadsheet analysis in Datama (or a connected host).
2. Export as HTML with Datama marker blocks.
3. In the PowerPoint add-in, click **Load HTML file** and select the export.

<br/>

# 3. Interact with the visualization

- Toggle **Edit mode** in editable PowerPoint views to adjust settings.
- Reload another file with **Load HTML file** at any time.
