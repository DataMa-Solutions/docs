> **Datama Spreadsheet combines spreadsheet flexibility with pivot-table analysis inside BI extensions.**

<br>

# 1. Pivot workflow

Datama Spreadsheet follows a pivot-table setup model based on three field areas:

## 1.1 Rows

Add categorical dimensions you want to list vertically in the pivot output.

## 1.2 Columns

Add dimensions you want to spread horizontally to compare categories side-by-side.

## 1.3 Values

Add numeric fields to compute business metrics in the pivot table.

<br>

# 2. Spreadsheet capabilities

Datama Spreadsheet includes the main spreadsheet capabilities available in Luckysheet, including:

## 2.1 Formatting

- Styling (font family, size, color, visual effects)
- Conditional formatting
- Text alignment and rotation
- Text truncation, overflow, automatic wrapping
- Data types (currency, percentages, decimals, dates, custom formats)
- Cell segmentation style (line breaks, sub/superscript, in-cell style)

## 2.2 Cells

- Drag and drop cell operations
- Fill handle and fill options
- Multiple selection
- Find and replace
- Merge cells
- Data validation (checkbox, drop-down list, date picker)

## 2.3 Rows and columns

- Hide, insert, delete rows and columns
- Freeze rows and columns
- Split text into multiple columns

## 2.4 Operations

- Undo and redo
- Copy, paste, cut
- Keyboard shortcuts
- Format painter
- Selection drag interactions

## 2.5 Formulas and functions

- Built-in formulas (math, text, date, financial, logical, lookup)
- Dynamic array and array formulas
- Remote and custom formulas

## 2.6 Tables and pivot tables

- Filters and multi-field sorting
- Pivot field arrangement (rows, columns, values, filters)
- Multiple aggregation methods in pivot results
- Drill-down by double-clicking pivot cells
